# Contributing

Thanks for helping improve Emitrace.

## Setup

1. Clone the repo.
2. Run `flutter pub get` in `emitrace/`.
3. Run the example app from `emitrace/example/`.

## Development Guidelines

- Keep changes focused and incremental.
- Avoid breaking existing public APIs.
- Add docs/examples for new user-facing behavior.

## Validation

Run before opening PR:

- `dart format .`
- `flutter analyze`
- `flutter test`

## Pull Requests

- Describe problem, approach, and screenshots (if UI changes).
- Mention backward compatibility impact.
- Update `CHANGELOG.md` for user-facing changes.
