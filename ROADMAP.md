# Roadmap

## Current Product Direction

Emitrace remains a local-first debug toolkit. No backend, auth, cloud storage, or AI summarization is required for core workflows.

## Near Term

- Better web/desktop parity for screenshot and report actions
- Optional JSON report export button in panel UI
- Improve network detail inspection UI for large payloads
- Add built-in redaction helpers for sensitive fields

## Mid Term

- Attach report files directly to Slack (bot token flow)
- Better router integrations (go_router helper docs)
- Timeline performance tuning for very large metadata payloads

## Long Term

- Report templates for QA teams
- Optional CI artifact export helpers
- Enterprise policy hooks for custom compliance checks
